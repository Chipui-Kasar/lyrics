# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in Tangkhul Lyrics, please report it privately to our security team.

### Contact Information

- **Email**: security@tangkhullyrics.com
- **Website**: https://tangkhullyrics.com/contact

## Supported Versions

We provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Security Best Practices

- Always keep your browser updated
- Use strong passwords for admin accounts
- Report suspicious activity immediately

Thank you for helping keep Tangkhul Lyrics secure!
